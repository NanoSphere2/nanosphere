# NanoSphere - Modern eCommerce & Web Solutions

Professional eCommerce services, web development, and automation solutions for modern startups.

![NanoSphere Preview](https://images.unsplash.com/photo-1551288049-bebda4e38f71?auto=format&fit=crop&q=80&w=1200&h=600)

## 🚀 Features

- **eCommerce Solutions:** Expert management for eBay, Amazon, Shopify, and Walmart.
- **Web Development:** Fast, modern, and responsive websites built with React, Vite, and Tailwind CSS.
- **Automation Systems:** Intelligent business process automation and AI integration.
- **Modern UI/UX:** Polished design with smooth animations using Framer Motion.
- **Real-time Booking:** Integrated meeting scheduling system with Firebase.
- **AI Assistant:** Smart chat assistant powered by Google Gemini.

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite, TypeScript
- **Styling:** Tailwind CSS 4, Lucide Icons
- **Animations:** Framer Motion
- **Backend/Database:** Firebase (Firestore, Auth)
- **AI:** Google Gemini API (@google/genai)
- **UI Components:** Base UI, shadcn/ui patterns

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd nanosphere
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up Firebase:
   - Create a `firebase-applet-config.json` in the root directory with your Firebase project credentials.
   - Alternatively, update `src/firebase.ts` to use environment variables.

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
